## 🧠 The RAG Engine — Built From Scratch

This isn't a wrapper around an AI API. The chatbot is a **complete Retrieval-Augmented Generation system**:

```
User Query → Tokenization → TF-IDF Vectorization → Cosine Similarity Search → Response Generation
```

**Key Features:**
- 📝 **TF-IDF Indexing** of 35+ knowledge base documents
- 🔎 **Cosine Similarity** retrieval for semantic matching
- ✍️ **Levenshtein Distance** fuzzy matching for typo tolerance (`hellow` → `hello`)
- 💬 **27+ Conversational Handlers** — greetings, farewells, emotions, meta-questions, ability checks, hiring queries, fun facts, tech definitions, and more
- 🧩 **Smart Fallback** — fuzzy keyword matching before defaulting
- 🎯 **Intent Detection** — routes queries to specialized formatters (projects, skills, education, contact, hire)

## 🚀 Quick Start

No build tools needed. Just serve the static files:

```bash
# Clone
git clone https://github.com/PAMIDIROHIT/portfolio.git
cd portfolio

# Option 1: Python
python3 -m http.server 8080

# Option 2: Node
npx serve .

# Option 3: Just open index.html in your browser
```

Then visit:
- **Homepage** → `http://localhost:8080`
- **AI Chatbot** → `http://localhost:8080/gemini.html`
- **Search** → `http://localhost:8080/search.html`
